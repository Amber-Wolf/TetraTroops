Ouya Project done in Fall 2013.
  
Everything outside of the Texample class is my original code or an extension of sample code. Uses JavaGl.
 
Compile like an Android project. The APK can be played in Ouya device. See more information at: https://www.ouya.tv/developer-kit/ for Ouya information and http://developer.android.com/tools/building/building-eclipse.html for compiling from the ADT or http://developer.android.com/tools/building/building-cmdline.html for compiling from the command line!
